\[latexpage\] A predicate, in first order logic, is a function that returns True or False. Let $P(n)$ be a predicate. If $P(0)$ is True and $forall n in mathbb{N} : (P(n) Rightarrow P(n+1))$ is True, then $forall n in mathbb{N}$ $P(n)$ is True.
